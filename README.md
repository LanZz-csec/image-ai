<h1  align="center"> Discord Image AI Bot </h1>

## 📝 | Tutorial
 
### 💪🏻 Non-Docker

> The `config.json` file should be configured first. Don't forget to change `BOT_TOKEN` & `REPLICATE_API_KEY`

Install all dependencies and deploy Slash Commands

```sh

npm  install

npm  run  deploy

```

Start the bot

```sh

node  index.js

```
